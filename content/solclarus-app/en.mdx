## Project Overview

A personal portfolio website showcasing my works and blog posts. Built with Next.js 14, it features dark mode, internationalization, and a clean, responsive design.

## Core Features

### Dark Mode & Internationalization

- Implemented dark mode using next-themes
- Multi-language support (English/Japanese) using next-intl
- Smooth theme transitions

### Blog System

- Integration with Newt CMS
- Markdown content support
- Syntax highlighting for code blocks

### Technical Architecture

- Next.js 14 App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Responsive design principles

## Development Process

The site was built with a focus on performance and user experience. Key considerations included:

- SEO optimization
- Accessibility standards
- Performance optimization
- Mobile-first approach
